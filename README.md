# Blackjack

This program we developed is made to similuate the experience of playing the authentic and classic casino game Blackjack.

Authors:
Marco Salazar, Nguyen Vo, Jary Castro

How to Use:
 Unzip the file in your prefered file directory
 make sure you have Python 3 installed and that you also have the tkinter module winthin Python
 open your IDE of choice
 open the Black--jack-test folder
 and run the main.py file a small window shoudl open on your pc, and you can play from there.

 Enjoy from there on!
